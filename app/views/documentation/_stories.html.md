## User Stories

Our user stories form the basis for our User Acceptance testing.

* As a system administrator... (inherits User)
	* I want to generate reports of pest sightings based on parameters I select.
	* I want to print reports to keep a hard copy.
	* I want to delete users to keep them from accessing the system.
	* I want to delete recorded pests so it no longer exists on the system. 
* As a user...
	* I want to login to the system so I can access it's features.
	* I want to report a specific pest to make others aware of it's presence.
	* I want to record a pest sighting to make others aware of it's status.
	* I want to update a pest sighting to keep others up to date on it's status.
	* I want to view all reported sightings to identify the status of a pest.
	* I want to filter through reported sightings to keep track of their status.
	* I want to be able to update my account information to keep it up-to-date.
